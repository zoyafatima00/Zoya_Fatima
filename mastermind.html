Master Mind

I developed this app at the end of 2023 to provide users with guided hypnosis and relaxation audio tracks.

As the sole developer, I focused on designing a user-friendly interface and integrating a diverse range of content.

The app allows users to access various hypnosis sessions for relaxation and personal growth, featuring options to browse content, save favorites, and track progress.

It includes customizable settings and session reminders for a tailored experience.

I worked on this project to offer an accessible tool for mental wellness, continuously updating it based on user feedback.

Technologies:
  • Flutter - For cross-platform mobile app development
  • Dart - Programming language used with Flutter
  • Firebase - For authentication, real-time database, and cloud functions
  • Firestore - Real-time data storage and management
  • Stripe or PayPal - Secure payment processing
  • RESTful APIs - For frontend and backend communication
  • Git - Version control
  • GitHub - Repository hosting and collaboration
  • Android Studio or Visual Studio Code - Development and code editing
