Social Spark

This app was developed as a semester project to provide users with a comprehensive social media experience. The app enables users to post updates, interact with content through likes and comments, and manage their social connections by following other users.

Key features include:

  • Post Creation: Users can create and share posts with text and images.
  • Likes and Comments: Users can engage with posts by liking and commenting. Comments can be added, edited, or deleted.
  • Update Captions: Users can update the captions of their posts at any time.
  • Follow Functionality: Users can follow others to see their updates in their feed.
The app uses Firebase for backend services, providing a robust and scalable solution for user authentication and real-time data synchronization. This eliminates the need for external APIs, allowing for a more streamlined development process.

The project aimed to create an engaging and user-friendly social media platform, offering a hands-on experience with Firebase integration and Flutter development.

Technologies:
  • Flutter - Utilized for developing a cross-platform mobile application, ensuring a native-like experience on both iOS and Android devices.
  • Dart - The programming language used with Flutter to write the app's code and manage its logic.
  • Firebase - Employed for user authentication, real-time data storage, and managing user interactions without relying on external APIs.
  • Firestore - A NoSQL database service from Firebase used to store and synchronize user data in real-time.
  • Git - Version control system used to manage and track changes in the project’s codebase.
  • GitHub - Platform for hosting the project repository, enabling collaboration and version control.
  • Android Studio or Visual Studio Code - Integrated development environments (IDEs) used for coding, debugging, and testing the app.

The choice of these technologies ensured a robust and scalable application, allowing for efficient development and deployment of the social media platform.
