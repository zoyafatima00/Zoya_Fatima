GoToApp - Connecting Customers and Vendors

I was part of the team that developed the KP Market app, which facilitates seamless interactions between customers and vendors.

My contributions focused on front-end development, ensuring an intuitive interface for browsing services, placing orders, and managing transactions.

I worked on integrating APIs to enable real-time notifications and enhance user experience.

I also assisted in implementing secure payment gateways to ensure safe and reliable transactions.

Our team's goal was to create a reliable and efficient platform for service marketplaces, continuously improving it based on user feedback.

Technologies:
  • Flutter: Built the cross-platform user interface for both Android and iOS.
  • Dart: Provided the programming foundation for the app's architecture.
  • Firebase: Used for user authentication and backend support.
  • Cloud Firestore: Ensured real-time database management and synchronization.
  • Stripe: Integrated to manage secure payment transactions.
  • RESTful APIs: Connected the front-end to back-end services.
  • Git: Managed version control for the development process.
  • GitHub: Facilitated code hosting and team collaboration.
  • Visual Studio Code: Utilized as the primary IDE for development and debugging.
